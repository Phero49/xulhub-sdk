# SDK Examples

Explore how to build different types of interactive exercises using XulHub SDK. These examples demonstrate real-world usage patterns.

## Available Examples

### [True or False](./true-or-false)
A simple exercise where students decide if a statement is true or false. Demonstrates:
- `onReady` mode handling.
- `saveContent` for editor configuration.
- `verifyAnswer` for grading.
- `createFileUploadArea` for adding images and audio.
